# SkillsBuild badges slides — outline

The submission form asks for a presentation link showing your team’s completed IBM SkillsBuild badges (images + links). Google Slides is preferred.

## Recommended slide structure

Slide 1 — Title
- IBM AI Racing League — SkillsBuild Badges
- University name
- Course name
- Team name
- Team members

Slide 2 — Badge summary
- Bullet list of badges completed (names)
- Date range completed

Slides 3+ — One badge per slide
For each badge:
- Badge name (large title)
- Screenshot of badge / completion page
- Verification link (clickable)
- Who completed it (member name)
- 1–2 bullets: what you learned + how it impacted the project

Final slide — How SkillsBuild impacted the submission
- 3 bullets connecting SkillsBuild learning → actual implementation decisions
- Link to repo + blog

## Sharing
Set the Slides sharing to: **Anyone with the link can view**.

## Where to store assets in this repo
Optional: put badge screenshots under `docs/skillsbuild/` so they’re easy to drag into Slides.
